# ICHITO v5.0.0 Release Notes

## Major Updates & New Features
- **Total Architecture Overhaul**: Complete system upgrade with modern state management.
- **Global Design Consistency**: All images across the app are now uniformly shaped (rectangular) with the active theme's corner radius and thin borders.
- **Enhanced Order Wizard**: 
  - Dynamic Grid/List toggles in all steps.
  - Measurement section now smartly pre-fills based on garment type and syncs with global user settings.
  - Added new "Garment Form" directly within the wizard.
- **Notification Engine**: 
  - Real-time instant alerts for every Create, Update, and Delete action for Clients, Orders, Fabrics, Designs, Garments, and Notes.
  - Smart due date reminders at 8 AM, 12 Noon, and 6 PM for order deadlines.
- **Localization Mastery**: 
  - 100% comprehensive Sheng (Street) translation, culturally adapted.
- **Improved UI/UX & Redesigns**:
  - A redesigned help section with categorized expansion tiles and a WhatsApp support integration.
  - Fully revamped Order Details with editable notes, polished timeline, and detailed financials tracking.
  - Comprehensive customer financial histories with precise filters.

## Bug Fixes & Optimizations
- Fixed keyboard clipping on various forms.
- Re-architected settings and security flow with robust biometric locking and simplified menus.
- Addressed minor routing and deep-linking issues in dashboard summary cards.
