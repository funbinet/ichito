# ICHITO

The complete tailor management system. Built with care for tailors everywhere.

"Ichito" means "to work" or "work/job" in Sheng (Kenyan slang).

## Features
- **Offline First**: Works entirely offline with a local SQLite database. No subscriptions, no internet required.
- **Customer Management**: Detailed customer profiles, measurements, loyalty tiers, and photo integration.
- **Order Wizard**: A guided 6-step flow for creating precise, error-free orders.
- **Financial Tracking**: Partial payments, balance tracking, and reporting.
- **Notes System**: Built-in specialized notes for General, Church (with Bible verse tagging), and Chama (group savings).
- **Security**: Robust PIN and Biometric app lock system.
- **Multi-language**: Seamless switching between English and authentic Sheng.
- **Garments & Fabrics Catalogs**: Customizable libraries of your offerings.

## Architecture
ICHITO is built with Flutter and Dart, utilizing a Clean Architecture approach with Provider for state management, SQLite (`sqflite`) for structured data, and Hive for persistent preferences.
