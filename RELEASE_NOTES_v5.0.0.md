# ICHITO v5.0.0 Release Notes

Welcome to ICHITO Version 5.0.0! This is a massive update focusing on deeper personalization, accessibility, and cultural localization, specifically tailored for our Kenyan users.

## What's New

### 🇰🇪 Comprehensive Sheng Localization
- **Instant Toggle:** The entire application now supports switching between English and authentic Kenyan Sheng. Every single label, button, notification, and system message translates instantly.
- **Authentic Slang:** We've carefully crafted translations using authentic street parlance (e.g., *Profili* for profile, *Maoda* for orders, *Wateja* for clients, *Munde* for total). 
- **Absolute Coverage:** Absolutely no English text remains when Sheng is selected, providing a fully immersive, culturally relevant experience.

### 🔍 Global Font Scaling for Accessibility
- **Adjustable Text Size:** Added a new global font size adjustment feature in the *Appearance Settings*.
- **Accessible Design:** Easily increase or decrease the font size across the entire application to ensure maximum readability for users who struggle with tiny text.

### 🖼️ Square Image Containers
- **Unified Visuals:** All image containers across the app (customers, fabrics, designs, and profiles) have been standardized to strict square containers with square cropping.
- **Corner Style Support:** The containers fully respect your selected theme's corner style (whether sharp, rounded, etc.), bringing a cohesive aesthetic without circular clipping.
- **Circular Previews:** Maintained a specific circular preview strictly in the client details view when editing an avatar, as requested, for seamless selection.

## Improvements & Fixes
- Enhanced the `LanguageProvider` with over 400 new translation keys.
- Streamlined settings persistence for font sizes via `SettingsRepository`.
- Refactored multiple UI components (`DesignCard`, `FabricCard`, `CustomerListTile`) to dynamically adapt to theme corner radii.

Thank you for choosing ICHITO! Keep tailoring!
